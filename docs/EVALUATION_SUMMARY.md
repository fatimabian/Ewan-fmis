# FMIS Evaluation Summary

**System:** Farmer Management Information System (FMIS)  
**Office:** Office for Agricultural Services, Rosario, Batangas  
**Evaluation date:** July 20, 2026  
**Respondents:** 2 (one client representative and one end user)

## System Usability Scale (SUS)

SUS was scored using the standard alternating-item calculation: for odd-numbered questions, subtract 1 from the response; for even-numbered questions, subtract the response from 5; multiply the total contribution by 2.5.

| Respondent type | SUS score |
|---|---:|
| End user | 75.0 |
| Client representative | 77.5 |
| **Mean** | **76.25 / 100** |

The results indicate favorable perceived usability for the two evaluated users. Because the sample contains only two respondents, the result should be reported as validation evidence from the designated client and end user, not as a population-wide statistical conclusion.

## Service-oriented evaluation

Both respondents selected 5 (Strongly Agree) for all five service-oriented statements.

| Measure | Result |
|---|---:|
| Responses analyzed | 10 |
| Mean rating | **5.00 / 5.00** |
| Strongly Agree responses | **100%** |

The evaluated users strongly agreed that FMIS improves service-delivery quality and efficiency, supports accurate and reliable agricultural services, meets the Office's needs, and contributes positively to farmer service delivery.

## Qualitative feedback and action taken

| Client feedback | Implementation response |
|---|---|
| Farmer records and activity-log reports were useful | Retained and covered by automated access and record tests. |
| Add filtered reports | Added a vertical report-filter panel with Year, Barangay, Commodity, Category, Period, Request Status, and Format. Region, Province, and Municipality filters were intentionally omitted because FMIS covers Rosario, Batangas only. |
| Add commodity per parcel | Added the Commodity per Farm Parcel report with farmer, parcel, barangay, commodity, area, schedule, planting date, and harvest date. |
| Add farmer statistics | Added active, female, male, senior, 4Ps, PWD, and Indigenous farmer indicators to Reports & Analytics. |
| Add remarks | Added an optional, 1,000-character internal remarks field to farmer registration, updates, detail views, and the Farmer Master List export. |
| “Very good!” | Recorded as positive additional feedback. |

One respondent left the qualitative section blank. The project does not invent or infer comments for that respondent.
